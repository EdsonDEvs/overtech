A Uber está mudando o foco na corrida pelos táxis autônomos

A Uber lançou uma nova divisão nomeada 'AV Labs' com o intuito de coletar dados de direção para futuros parceiros de robotáxis. A empresa, que anteriormente trabalhava no desenvolvimento de seus próprios veículos autônomos, agora aposta na coleta e oferta de dados para este mercado emergente.

Os dados como nova forma de contribuição para os taxis autônomos

A Uber acredita que, ao coletar e oferecer uma quantidade maior de dados, pode auxiliar seus parceiros de veículos autônomos a resolver os casos mais estranhos ou incomuns que podem ocorrer. Esta é uma abordagem diferente para a empresa, que anteriormente se concentrava no desenvolvimento de suas próprias soluções autônomas.
